Krótki opis 
-------------------
Strona służy do zaprezentowania posiadanych przeze mnie umiejętności z zakresu technologii webowych. Została wykonana w jednych z początkowych miesiącach mojej nauki.

Użyte technologie 
--------------------
- HTML5,
- CSS3,
- RWD (media queries),
- Bootstrap,
- jQuery:
